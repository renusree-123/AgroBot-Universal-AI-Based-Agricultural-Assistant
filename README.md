# 🌾 AgroBot – Universal AI Based Agricultural Assistant

AgroBot is an AI-powered assistant developed as part of the Infosys Springboard Internship Program.  
The system helps farmers by providing intelligent agricultural guidance using Natural Language Processing and AI models.

---

## 📌 Milestone 1 Features
- User Registration & Login
- Flask Web Application Setup
- Gemini API Integration
- Chat Interface UI
- Modular Project Structure

---

## 🛠 Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Jinja2 Templates
- Google Gemini API

1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run app

## 🎯 Objective
To build an AI based smart agriculture assistant that helps farmers with crop suggestions, disease detection, and farming guidance.
